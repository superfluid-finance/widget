## WalletManager

An interface for interacting with a wallet management library (e.g. Web3Modal, RainbowKit, ConnectKit etc).

| Property | Type | Value | Description |
| ----------- | ----------- | ----------- | ----------- |
| isOpen | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | N/A | Is the wallet connecting view open? If it is, we might want to hide the widget. |
| open | [any](https://www.typescriptlang.org/docs/handbook/2/everyday-types.html#any) | N/A | N/A |
