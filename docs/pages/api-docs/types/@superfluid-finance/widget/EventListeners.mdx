## EventListeners

A set of non-blocking callback functions that are triggered in response to the widget events.

| Property | Type | Value | Description |
| ----------- | ----------- | ----------- | ----------- |
| onButtonClick | (props: \{ type: 'next_step' \| 'skip_step' \| 'step_label' \| 'connect_wallet' \| 'switch_network' \| 'invoke_transaction' \| 'back_transactions' \| 'success_button' \| 'superfluid_dashboard' \| 'retry_gas_estimation' \| 'force_invoke_transaction' \| 'skip_to_next' \| 'copy_account_address' \| 'view_transaction_on_block_explorer' }) => void | N/A | Called when a button is pressed. Usually a call to action (CTA). |
| onPaymentOptionUpdate | (paymentOption: PaymentOption) => void | N/A | Called when the payment option is initialized or changed by the user. |
| onRouteChange | (props: \{ route: 'transactions' \| 'step_payment_option' \| 'step_wrap' \| 'step_review' \| 'success_summary' }) => void | N/A | Called when the widget route changes. "Route" is a term to define the _view_ the user sees. |
| onSuccess | (...args: unknown\[]) => void | N/A | Called when the checkout is successfully finished. |
| onSuccessButtonClick | (...args: unknown\[]) => void | N/A | Called when the merchant's success button is defined in the schema and it's clicked. |
| onTransactionSent | (props: \{ functionName: 'createFlow' \| 'deleteFlow' \| 'grantPermissions' \| 'revokePermissions' \| 'setFlowrate' \| 'setFlowrateFrom' \| 'updateFlow' \| 'updateFlowOperatorPermissions',hash: 0x\$\{string} }) => void | N/A | Called when a transaction is executed. |
