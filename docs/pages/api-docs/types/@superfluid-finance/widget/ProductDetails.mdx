## ProductDetails

The details of the checkout product.

| Property | Type | Value | Description |
| ----------- | ----------- | ----------- | ----------- |
| description | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | ... | The description of the checkout product. |
| imageURI | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | ... | The image URI of the checkout product. Can be an URL, can be a data URI (e.g. inline Base64). |
| name | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | ... | The name of the checkout product. |
| successText | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | ... | Text shows on the success button in the final screen of the checkout. |
| successURL | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | ... | The URL to redirect to when the success button is clicked. |
